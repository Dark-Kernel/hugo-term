Install .deb in Arch

2025-06-25Sumit Patel
#deb  #arch 
  1. It’s just an archive
bsdtar -xf "package_0.1.0-1_amd64.deb" data.tar.xz
tar xf data.tar.xz

[–more–]

this will give usr directory containing binary in usr/bin/ and other data in usr/share
