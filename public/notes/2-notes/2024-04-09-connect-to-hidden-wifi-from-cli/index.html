connect to hidden wifi from cli

2024-04-09Sumit Patel
#wifi 
  1. Install iwd for iwctl
sudo pacman -S iwd

[–more–] 2. iwctl 3. Enter following info

“station connect-hidden <“network name”>”

e.g

> station wlan0 connec-hidden "<SSID>"
